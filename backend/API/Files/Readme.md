## Cheatsheet

### Regex

#### Empty line 
`^(?:[\t ]*(?:\r?\n|\r))+`

#### Find word with exact number of letters:

- 0 to 4 lettered words `^.{0,4}$`
- 6 to infinity lettered words `^.{6,}$`
